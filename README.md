

# 🎵 Musical Stage – Spotify Clone (Frontend)

> A modern and responsive Spotify-inspired music streaming UI built using **HTML, CSS, and JavaScript**.

---

## 🚀 Live Demo

🔗 https://github.com/Umangi-webdev/Spotify-clone---Musical-Stage/tree/main

## 📌 Overview

**Musical Stage** is a frontend clone of Spotify that replicates the look and feel of a real music streaming platform.
This project focuses on creating an engaging UI/UX with smooth interactions and responsive design.

## ✨ Features

* 🎧 Beautiful Spotify-inspired UI
* 📱 Fully responsive design (Mobile + Desktop)
* 🎵 Music player interface (Play/Pause functionality)
* 📂 Playlist section design
* 🔍 Search bar UI
* 🎨 Smooth animations and transitions
* ⚡ Fast and lightweight frontend

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & Layout (Flexbox / Grid)
* **JavaScript (Vanilla JS)** – Functionality
